치유미 개인정보 처리방침

최종 수정일: 2026-06-22

1. 수집하는 정보

치유미(이하 "봇")는 다음 정보를 수집·저장합니다.

  • 디스코드 사용자 ID (고유 식별자)
  • 치유미코인 보유량 및 일일 지급 이력
  • 채팅·음성 활동으로 누적되는 경험치, 레벨, 음성 채널 체류 시간(서버 구분 없이 계정 단위로 누적)
  • 이용약관 동의 여부 및 동의 시각
  • 서버 관리자가 설정한 입퇴장 알림 채널 및 문구
  • 서버 관리자가 욕설 검열을 켠 경우, 삭제된 메시지의 작성자·채널·내용(설정된 로그 채널에만 전송, 별도 영구 저장 없음)
  • /문의로 제출한 신고·피드백·버그 신고 내용(작성자, 서버 이름, 서버 주소, 내용)

/전적검색 사용 시 입력한 라이엇 ID(닉네임#태그)는 Riot Games API 조회에만 사용되며, 조회 결과는 최대 10분간 메모리에 임시 보관된 뒤 자동으로 삭제됩니다.

/순위 명령어 사용 시 카드 이미지를 만들기 위해 디스코드에 등록된 프로필 사진을 디스코드 CDN에서 불러오며, 별도로 저장하지 않습니다.

봇은 닉네임, 일반 메시지 내용(욕설 검열 대상이 아닌 경우) 등 그 외의 개인정보를 별도로 수집하지 않습니다.

2. 정보 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다.

  • 치유미코인 잔액 관리(조회, 지급, 베팅)
  • 레벨·순위 기능의 경험치/음성 시간 집계 및 카드 이미지 생성
  • 입퇴장 알림 메시지 전송
  • 이용약관 동의 여부 확인
  • 욕설 검열 기능의 메시지 삭제 및 관리자 로그 기록
  • 전적검색 기능의 전적 조회
  • 신고·피드백·버그 신고 내용의 운영자 전달

3. 제3자 제공

/전적검색 사용 시 입력한 라이엇 ID는 전적 조회를 위해 Riot Games의 API 서버로 전달됩니다. 그 외 정보는 제3자에게 제공되거나 판매되지 않습니다.

4. 정보 보관 및 위치

데이터는 봇이 실행되는 서버의 로컬 저장소(JSON 파일)에 보관되며, 전적검색 조회 결과는 최대 10분간 메모리에만 임시 보관됩니다.

5. 정보 삭제

사용자는 서버 관리자 또는 운영자에게 요청하여 본인의 데이터 삭제를 요청할 수 있습니다.

6. 정책의 변경

본 처리방침은 사전 통지 없이 변경될 수 있습니다. 변경된 내용은 본 페이지에 게시된 시점부터 적용됩니다.
