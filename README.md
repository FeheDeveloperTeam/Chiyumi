# Chiyumi

Chiyumi는 디스코드 서버 운영을 돕기 위해 제작된 Node.js 기반 디스코드 봇입니다.

서버 관리자가 인증 메시지를 등록하면, 서버 사용자는 버튼을 눌러 지정된 역할을 받을 수 있습니다. 이를 통해 신규 사용자의 서버 입장 절차를 더 간단하고 명확하게 구성할 수 있습니다.

## 주요 기능

- 버튼 기반 서버 인증 기능
- 인증 완료 사용자에게 역할 자동 지급
- 관리자 권한 기반 인증 메시지 생성
- 디스코드 슬래시 명령어 지원

## 개발 정보

- Node.js
- discord.js

## 안내

이 저장소는 Chiyumi 봇의 개발 및 관리용 저장소입니다. 봇의 기능은 서버 운영 목적에 맞춰 계속 확장될 수 있습니다.
