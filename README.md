# Chiyumi

Chiyumi는 디스코드 서버 운영과 놀이 기능을 함께 제공하는 Node.js 기반 디스코드 봇입니다.

## 주요 기능

- **인증** (`/인증`): 버튼을 눌러 인증하면 지정한 역할을 자동으로 지급
- **공지** (`/공지`): 관리자가 지정한 채널에 임베드 공지 전송
- **로그** (`/로그`): 메시지 삭제/수정, 음성 입퇴장 등 서버 활동을 채널에 기록
- **입퇴장 알림** (`/입퇴장`): 멤버 입장/퇴장 시 커스텀 문구로 채널에 알림
- **검열** (`/검열`): 욕설검열·도배검열을 각각 켜고 꺼서 채팅에서 욕설이나 도배를 자동 감지해 삭제하고 로그 채널에 기록
- **티켓** (`/티켓`): 관리자가 티켓 채널과 안내 문구를 설정하면 채널에 "티켓 생성" 버튼이 올라가고, 누르면 비공개 스레드가 생성되어 티켓을 연 사람과 서버 관리자만 볼 수 있음. 스레드 안에는 관리자 전용으로 사용자 추가, 닫기(관리자 외 열람 권한 제거), 대화 내용 txt 저장, 삭제 버튼 제공
- **출석** (`/출석`): 하루에 한 번 출석하면 치유미코인과 경험치 지급
- **치유미코인** (`/코인`): 잔액 조회, 채팅/음성 활동으로도 코인 획득
- **도박** (`/도박`): 슬롯머신, 홀짝, 숫자맞추기, 블랙잭, 가위바위보를 코인을 걸고 플레이
- **키우기** (`/키우기`): 내 고양이에게 밥 주기·씻겨주기·놀아주기로 애정도를 관리하고, 실제 입양일로부터 흐른 시간에 따라 새끼 고양이부터 전설의 고양이까지 성장. 각 보살핌 행동은 하루에 한 번, 성공/실패가 랜덤으로 갈림
- **순위** (`/순위`): 채팅/음성 활동으로 쌓은 레벨·경험치를 카드 이미지로 확인, 서버별 채팅·음성 순위 페이지네이션 제공. 관리자는 레벨업 알림 채널과 문구도 커스텀 설정 가능
- **전적검색** (`/전적검색`): 리그 오브 레전드 닉네임으로 최근 전적, 라인별 매치업, 티어 엠블럼 조회 (Riot Games API)
- **문의** (`/문의`): 신고/피드백/버그 신고를 버튼과 모달로 접수해 지정 채널로 전달
- **도움말** (`/도움말`): 카테고리별 드롭다운으로 전체 명령어 목록 안내
- **핑** (`/핑`): 봇의 응답 속도 확인
- **개발자** (`/개발자`, 개발자 전용): 사용자 ID 기반으로 봇 이용제한·해제·제한 여부 확인
- 모든 명령어는 디스코드 슬래시 명령어로 제공되며, 처음 사용 시 이용약관/개인정보 처리방침 동의 절차를 거칩니다

## 개발 정보

- Node.js
- discord.js
- @napi-rs/canvas (순위 카드 이미지 생성)
- googleapis (구글 시트 데이터 백업/뷰어 동기화, `.env`에 `GOOGLE_SHEET_ID`/`GOOGLE_SERVICE_ACCOUNT_EMAIL`/`GOOGLE_SERVICE_ACCOUNT_PRIVATE_KEY` 설정 시에만 동작)
